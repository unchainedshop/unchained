## Fetch 50 Products with Breadcrumbs and Asset (Product List):

Vendure:

```
{
  products(options: { take: 50 }) {
    totalItems
    items {
      id
      name
      description
      collections {
        breadcrumbs {
          name
        }
        name
      }
      assets {
        id
        source
        name
      }
    }
  }
}
```

Result: ~ 115-180ms (uncached)

Unchained:

```
{
  productsCount
  products(limit: 50) {
    _id
    texts {
      _id
      title
      description
    }
    assortmentPaths {
      links {
        assortmentTexts {
          title
        }
      }
    }
    media {
      _id
      file {
        _id
        url
        name
      }
    }
  }
}
```

Result: ~ 30-45ms (uncached)
