# Delivery Plugins

Delivery plugins handle shipping and fulfillment methods.

| Adapter Key | Description |
|-------------|-------------|
| [`shop.unchained.post`](./delivery-post.md) | Generic postal delivery |
| [`shop.unchained.delivery.send-message`](./delivery-send-message.md) | Message-based delivery (digital) |
| [`shop.unchained.stores`](./delivery-stores.md) | In-store pickup |

## Creating Custom Delivery Plugins

See [Custom Delivery Plugins](../../extend/order-fulfilment/fulfilment-plugins/delivery.md) for creating your own delivery adapters.
