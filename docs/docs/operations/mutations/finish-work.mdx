Register a work attempt manually.
Note: Usually, work attempts are handled internally by the inbuilt cron
worker. This mutation is part of the interface for "outside" workers.


```graphql
finishWork(
  workId: ID!
  result: JSON
  error: JSON
  success: Boolean
  worker: String
  started: Timestamp
  finished: Timestamp
): Work!
```
