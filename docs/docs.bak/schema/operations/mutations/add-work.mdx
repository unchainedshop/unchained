Add work to the work queue. Each type has its own input shape. If you pinpoint the worker by setting it
during creation, the work will be only run by the worker who identifies as that worker.


```graphql
addWork(
  type: WorkType!
  priority: Int! = 0
  input: JSON
  originalWorkId: ID
  scheduled: Timestamp
  retries: Int! = 20
  worker: String
): Work
```
